Assignments for the first week of the course.

# Home work 1

```python
# Number 01
n = int(input("Enter a number: "))
count = 1
sum = 0
while count <= n:
    sum += 1 / count
    if count == n:
        print("1/", count, end=" = ")
    else:
        print("1/", count, end=" + ")
    count += 1
print(sum)

```

# Home Work 2

```python
# Number 02
n = int(input("Enter a number: "))
count = 1
sum = 0
while count <= n:
    sum += 1 / (count**2)
    if count == n:
        print("1/", count**2, end=" = ")
    else:
        print("1/", count**2, end=" + ")
    count += 1
print(sum)

```
